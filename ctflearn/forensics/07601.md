Running the command
```bash
binwalk AGT.png
```
reveals that there is hidden data inside the image. Extract it using
```bash
binwalk -e AGT.png
```
Locate the second image by going to _AGT.png.extracted > Secret Stuff... > Don't Open This... and run the command
```bash
strings "I Warned You.jpeg" | grep "CTF"
```
to reveal the flag.
